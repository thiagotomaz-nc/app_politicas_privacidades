Política de Privacidade

CalculaIFRN

Última atualização: 2026

Esta Política de Privacidade descreve como o aplicativo CalculaIFRN trata as informações dos usuários. O aplicativo foi desenvolvido como uma ferramenta educacional para auxiliar estudantes no cálculo de médias acadêmicas.

1. Desenvolvedor

O aplicativo CalculaIFRN foi desenvolvido por Thiago Tomaz de Araujo.

O projeto possui caráter educacional e acadêmico, podendo ser estudado, modificado ou aprimorado por estudantes da área de Tecnologia da Informação para fins de aprendizado, pesquisa ou desenvolvimento.

2. Finalidade do Aplicativo

O CalculaIFRN tem como objetivo auxiliar estudantes no cálculo de médias acadêmicas de forma simples e rápida.

O aplicativo foi inicialmente desenvolvido com foco em estudantes que utilizam sistemas de avaliação semestrais.

Futuras atualizações poderão incluir suporte para outros formatos de avaliação, como:

  • Sistemas bimestrais
  • Outros modelos acadêmicos utilizados por instituições de ensino

3. Coleta de Dados

O aplicativo não coleta, armazena ou compartilha dados pessoais dos usuários.

Todas as informações inseridas no aplicativo são utilizadas apenas localmente no dispositivo do usuário com a finalidade exclusiva de realizar cálculos de médias acadêmicas.

Nenhuma informação é enviada para servidores externos.

4. Uso de Anúncios

O aplicativo CalculaIFRN não contém anúncios, propagandas ou qualquer forma de monetização.

O aplicativo é disponibilizado gratuitamente para uso educacional.

5. Público-Alvo

O aplicativo foi desenvolvido principalmente para estudantes que desejam calcular médias acadêmicas de forma rápida e prática.

Embora seja direcionado a estudantes, o aplicativo pode ser utilizado por qualquer pessoa interessada em realizar cálculos de médias escolares.

O aplicativo não coleta dados pessoais de usuários de qualquer faixa etária.

6. Segurança das Informações

Como o aplicativo não coleta ou armazena dados pessoais, não há armazenamento ou processamento de informações sensíveis.

Todos os cálculos são realizados localmente no dispositivo do usuário.

7. Referência às Normas Acadêmicas

O aplicativo CalculaIFRN foi desenvolvido por Thiago Tomaz de Araujo como uma ferramenta educacional de apoio ao cálculo de médias acadêmicas.

As regras utilizadas para determinar situações como aprovação, reprovação e prova final são baseadas nas normas acadêmicas adotadas pelo Instituto Federal do Rio Grande do Norte (IFRN).

Este aplicativo não possui vínculo institucional oficial com o IFRN, sendo apenas uma ferramenta auxiliar desenvolvida de forma independente.

8. Alterações nesta Política

Esta Política de Privacidade poderá ser atualizada periodicamente para refletir melhorias no aplicativo ou mudanças em suas funcionalidades.

Recomenda-se que os usuários revisem esta página periodicamente para se manterem informados.
