POLÍTICA DE PRIVACIDADE – Anotei · Meus Gastos

Última atualização: março de 2026

1. Informações Gerais

O aplicativo Anotei · Meus Gastos tem como objetivo auxiliar o usuário no controle de despesas pessoais.

A privacidade e a segurança das informações são prioridades. O aplicativo foi desenvolvido para funcionar com armazenamento local, sem coleta de dados pessoais em servidores externos.

2. Coleta de Informações

O aplicativo não coleta dados pessoais do usuário.

  • Não solicita nome, CPF, e-mail ou telefone
  • Não exige criação de conta
  • Não envia informações para servidores externos
  • Não armazena dados financeiros fora do dispositivo

Todos os registros de despesas (valor, data e descrição) são armazenados localmente no dispositivo do usuário.

3. Segurança dos Dados

A segurança das informações depende principalmente das configurações do dispositivo do usuário.

  • Senha ou PIN do celular
  • Biometria ou reconhecimento facial

O aplicativo não possui sistema próprio de autenticação.

4. Serviços de Publicidade

O aplicativo pode utilizar serviços de publicidade fornecidos por plataformas como Google Ads ou Unity Ads.

Esses serviços podem coletar automaticamente informações técnicas para exibição de anúncios, incluindo:

  • Identificador de publicidade do dispositivo
  • Endereço IP
  • Modelo do dispositivo
  • Sistema operacional
  • Dados de interação com anúncios

A coleta e o tratamento desses dados seguem as políticas de privacidade das respectivas plataformas de publicidade.

5. Compartilhamento de Dados

O aplicativo não compartilha dados pessoais com terceiros.

Dados eventualmente coletados por serviços de anúncios são tratados diretamente pelos provedores de publicidade.

6. Armazenamento e Retenção de Dados

  • Os dados das despesas são armazenados somente no dispositivo do usuário
  • Não há sincronização com nuvem
  • Os dados permanecem no aparelho até remoção pelo usuário ou desinstalação do aplicativo

7. Direitos do Usuário

De acordo com a Lei Geral de Proteção de Dados (LGPD), o usuário pode:

  • Solicitar informações sobre o tratamento de dados
  • Solicitar exclusão de dados armazenados no dispositivo

Como o aplicativo não mantém servidores de armazenamento, a exclusão pode ser realizada desinstalando o aplicativo.

8. Cookies e Tecnologias Similares

O aplicativo não utiliza cookies diretamente. Eventuais tecnologias de rastreamento são utilizadas apenas por serviços de publicidade integrados.

9. Responsabilidade do Usuário

  • Manter o dispositivo seguro
  • Utilizar bloqueio de tela
  • Não compartilhar acesso ao aplicativo

10. Alterações na Política

Esta política pode ser atualizada a qualquer momento para adequação legal ou melhoria do serviço.

Recomenda-se revisar esta página periodicamente.
